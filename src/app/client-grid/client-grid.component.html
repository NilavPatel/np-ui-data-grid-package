Client side simple data grid example.

To set client side operations in datagrid, add columns and dataSource in grid defination.

dataSource is typeof BehaviorSubject<DataSource>.

Retrive all data and pass those data to dataSource with "next()" method.

DataSource class has below properties:

  • data (Array of all data, total will be calculated from length of data for client side)
  • total (Only required in server side operations, for client side pass 0)
  • summary (Pass summary data object if showSummary is set to true)
  • isAllPages (This will be used for server side export to CSV functionality, default value is false)
