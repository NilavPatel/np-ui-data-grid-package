Client side simple data grid example.

To set client side operations in datagrid, add columns and datasource in grid defination.
datasource is typeof BehaviorSubject< DataSource >.
Retrive all data and pass those data to behaviourSubject with next method.
DataSource has below properties:
  • data (Array of data)
  • total (Only required in server side operations, for client side pass 0)
  • summary (Pass summary data object if isShowSummary is set to true)
  • isAllPages (This will be used for server side export to excel functionality, default value is false)
