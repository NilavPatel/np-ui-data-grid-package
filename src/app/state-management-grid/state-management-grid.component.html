State management Example

To set state management enable in data grid, set showToolBar to "true" and enableStateStoring to "true"

APIs for state management

To get all state use getAllState(). Which will return array of State object.

To set state list use setAllState(states: State[]). pass array of State object to set them in dropdown.

State class contains below properties:

  • name: string
  • columns: Column[]
