Toolbar Example

To show tool bar in data grid, set showToolBar to "true" in grid defination.

To show show Column Chooser in tool bar, set showColumnChooser to "true".

Column chooser will help to show/hide columns in data grid.

If any column has filterEnable set to "true", then toggle filter row button will be visible in tool bar.

For Export to csv:

To show export to CSV button in tool bar, set allowExportToCSV to "true".

In export to CSV in client side, it will download file with all data.

In server side it will make an extra call to onLoadData(option:LoadOptions) method.

In this method's option parameter, isAllPages is passed as true, which needs to be pass in "next()" method of dataSource as "isAllPages = true".
