Columns Reorder/Resize and other column properties

Column Resize

To add column resize set allowColumnResize to "true" in grid defination.

To resize column, drag at the right end of column in left or right direction.(On hover of right end of column cursor will be turned to "col-resize").

Column Reorder

To add column reorder set allowColumnReorder to "true" in grid defination.

To set column reorder columns, showToolBar and showColumnChooser must be set to "true"

To reorder columns, open column chooser and drag "move" icon up and down.

Right Aligned Text

To set right aligned text in cells, set rightAlignText to "true" in column defination.

Add CSS class to Cell

To add CSS class in cells, set styleClass with class name in column defination.

Toggle column

To show/hide column set visible in column defination.

width

To set width of column, set width of column defination. Pass only width in numbers. No need to pass "px" as this will be consider in pixel only.

onCellClick API

Bind onCellClick with any function, which will be callback when any column is clicked.

in onCellClick function, event, column and rowData is passed as parameters.

Sticky Columns

To set left hand column as sticky, set stickyColumnLeft to "true" in column defination.

To set right hand column as sticky, set stickyColumnRight to "true" in column defination.

Only one column allowed to be sticky at lest and right.

format

Only used for date type column, set format like 'dd/MM/yyyy' to set format of column.
