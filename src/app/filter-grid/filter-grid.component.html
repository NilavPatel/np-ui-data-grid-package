Filters Example.

To add filters in data grid set filterEnable to "true" in column definations.

To toggle filters, set showFilters to "true" or "false" in grid defination. Show filters

Default value for showFilters is "true".

Client Side Filering:

In Client side data grid all filter oprations will be done on client side.

Server Side Filering:

In Server side data grid filterColumns will be passed to onLoadData(options:LoadOptions). Which will contains array of all filter columns as {{ '{' }} column : string, filterOperator: string, filterValue: string, dataType: string {{'}'}}

Possible values for filterOperator are startswith, endswith, contains, gt, lt, ge, le, eq, ne.

Possible values for dataType are number, string, date, boolean.
